# 玄成接单系统 · 云服务器部署指南

把系统部署到你的云服务器，让你俩和客户随时随地能用。
建议系统：CentOS 7+/Alibaba Cloud Linux 或 Ubuntu 20.04+。下面以 root 操作。

---

## 0. 先在本机打好包，再上传

```bash
# 后端 jar（在项目根的 server 目录）
cd server && mvn clean package -DskipTests
#  产物：server/ruoyi-admin/target/ruoyi-admin.jar

# 前端静态（在项目根）
npm run build
#  产物：dist/  （整个文件夹）
```

需要上传到服务器的东西：
- `server/ruoyi-admin/target/ruoyi-admin.jar`
- `dist/`（整个文件夹）
- `deploy/xc_jiedan.sql`、`deploy/nginx.conf`、`deploy/application-druid.yml`、`deploy/ruoyi.service`

可用 `scp` 或宝塔面板上传。

---

## 1. 安装运行环境

```bash
# JDK 17
yum install -y java-17-openjdk        # Ubuntu: apt install -y openjdk-17-jdk
# MySQL 8、Redis、Nginx（用你习惯的方式装；宝塔面板可一键装）
# 确认：
java -version
mysql --version
redis-cli ping      # 返回 PONG
nginx -v
```

> 没经验推荐装**宝塔面板**（bt.cn），MySQL/Redis/Nginx 点几下就装好，再按下面步骤放文件即可。

---

## 2. 建库并导入数据

```bash
mysql -uroot -p -e "CREATE DATABASE xc_jiedan DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;"
mysql -uroot -p xc_jiedan < /path/to/xc_jiedan.sql
```
导入后已包含：登录用账号 `admin/admin123`、接单表、7 条示例订单、推送设置。
想要空数据开张：`TRUNCATE jiedan_order; TRUNCATE jiedan_timeline;`

---

## 3. 部署后端

```bash
mkdir -p /opt/xuancheng
# 上传 ruoyi-admin.jar 到 /opt/xuancheng/
# 上传 deploy/application-druid.yml 到 /opt/xuancheng/ ，编辑里面的 MySQL 密码
vi /opt/xuancheng/application-druid.yml

# 用 systemd 托管（开机自启 + 崩溃自拉起）
cp deploy/ruoyi.service /etc/systemd/system/ruoyi.service
systemctl daemon-reload
systemctl enable ruoyi
systemctl start ruoyi
systemctl status ruoyi        # 看到 active(running) 即可
journalctl -u ruoyi -f        # 实时日志，看到 “Started RuoYiApplication” 成功
```

---

## 4. 部署前端

```bash
mkdir -p /var/www/xuancheng
# 把 dist/ 里的所有文件上传到 /var/www/xuancheng/
cp deploy/nginx.conf /etc/nginx/conf.d/xuancheng.conf
vi /etc/nginx/conf.d/xuancheng.conf      # 改 server_name 为你的域名/公网IP
nginx -t && systemctl reload nginx
```

---

## 5. 放行端口（关键的安全设置）

- 云服务器**安全组**：只开放 **80**（如配 HTTPS 再开 **443**）。
- **千万不要**把 **8080 / 3306 / 6379** 对公网开放——它们只给本机用（nginx 反代 8080，MySQL/Redis 本机访问）。

---

## 6. 验证

浏览器打开 `http://你的域名或IP` → 登录 `admin / admin123` → 进入工作台。
- 客户提交链接：`http://你的域名/#/intake`
- 微信推送：登录后到「设置」页填两人的 Server酱 SendKey。

---

## 7. 上线后务必做的安全项

1. **改掉 admin 默认密码**（登录若依官方后台，或直接改 sys_user 密码）。
2. 确认 8080/3306/6379 未对公网开放（第 5 步）。
3. **配 HTTPS**（强烈建议，微信里打开链接也更稳）：
   ```bash
   yum install -y certbot python3-certbot-nginx
   certbot --nginx -d 你的域名
   ```
4. 中国大陆服务器用 80/443 + 域名需**已备案**；没备案可先用 `http://公网IP` 临时访问。

---

## 8. 以后更新版本

```bash
# 后端改了：本机 mvn package → 传新 jar → systemctl restart ruoyi
# 前端改了：本机 npm run build → 传新 dist 覆盖 /var/www/xuancheng → 无需重启
```

## 9. 数据备份（建议每天）

```bash
# 加到 crontab：每天凌晨 3 点备份
0 3 * * * mysqldump -uroot -p'你的密码' xc_jiedan > /opt/xuancheng/backup/xc_jiedan_$(date +\%F).sql
```
